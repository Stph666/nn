## 项目简介
本项目依托 **CARLA 0.9.14 开源自动驾驶仿真平台**，以Python为开发语言，融合**传统机器视觉**与**PID闭环控制算法**，实现了L2级智能辅助驾驶的全流程闭环验证，覆盖城市道路、高速等典型场景，以及晴天、雨天、雾天、黑夜等多天气状况，规避了真实道路测试成本高、风险大、场景难复现的问题。

项目采用**纯视觉无模型方案**，无需依赖深度学习模型，仅通过OpenCV实现多类交通元素识别，配合PID控制完成车辆自动行驶、预警与避让，可稳定实现车道保持、定速巡航、安全预警与紧急制动，完整还原了智能辅助驾驶系统的核心感知-决策-控制链路，适合课程设计、毕业设计与自动驾驶入门实践。

## 核心功能特点
### 1. 多模态环境感知系统
- **精准交通信号识别**：基于HSV颜色空间分割，限定ROI区域过滤背景干扰，可稳定识别红/黄/绿三色信号灯，实现红灯停车、绿灯通行、黄灯预警的交规逻辑。
- **多类交通标志识别**：复用视觉识别框架，支持停车牌、限速牌（30/50/60km/h）的识别，可自动触发强制停车或调整巡航车速。
- **行人检测与AEB紧急制动**：优化行人肤色HSV阈值，结合轮廓面积过滤解决误检问题，仅对真实CARLA行人触发制动，为最高优先级控制逻辑。
- **车道偏离预警（LDW）**：通过CARLA车道入侵传感器，实时监测车道线压线行为，触发屏幕红色大字警告与音频提示音。
- **前向碰撞预警（FCW）与障碍物避让**：基于车辆位置向量计算前方障碍物距离，分级触发减速、制动逻辑，避免追尾事故。
- **盲区监测预警（BSD）**：监测车辆侧方盲区状态，提升变道场景的行驶安全性。
- **天气自适应车灯控制**：通过仿真环境参数判断昼夜/雨雾状态，自动切换近光灯、雾灯，适配不同光照环境。

### 2. 稳定闭环车辆控制逻辑
- **PID定速巡航**：基于增量式PID控制器，实现目标车速的稳定闭环控制，可动态响应限速标志调整车速，抗坡度、逆风干扰能力强。
- **车道居中保持**：以CARLA路点为参考，通过横向偏移量计算修正方向盘角度，实现无偏移车道居中行驶。
- **多级安全控制优先级**：设计严格的控制优先级，保障行驶安全：
  `行人AEB > 停车牌 > 红绿灯 > 障碍物避让 > 定速巡航`，高优先级指令可直接覆盖低优先级操作。

### 3. 全场景仿真验证与交互体验
- **多场景适配**：支持CARLA内置所有城市地图，可在直道、弯道、十字路口、环岛等多种路况下稳定运行。
- **多天气测试**：支持晴天、雨天、雾天、黑夜等多种天气切换，验证车灯控制、信号识别的环境鲁棒性。
- **可视化交互界面**：基于Pygame实现实时渲染，支持多视角切换、状态信息显示、运行日志输出，调试便捷直观。
- **轻量易部署**：无复杂深度学习依赖，仅需基础Python与CARLA环境，一键启动即可运行，适配普通Windows电脑配置。

## 运行环境与安装步骤
### 基础环境要求
| 项目 | 版本要求 | 说明 |
| :--- | :--- | :--- |
| Python | 3.7 / 3.8 | 与CARLA PythonAPI版本兼容 |
| 仿真平台 | CARLA Simulator 0.9.14 | 官方稳定版，支持所有内置地图与天气 |
| 核心依赖库 | pygame==2.6.1、opencv-python==4.5.5.62、numpy==1.21.6 | 视觉识别、渲染与数值计算依赖 |
| 系统支持 | Windows 10 / 11 | 支持CARLA 0.9.14的官方推荐系统 |

### 完整安装流程
1.  **克隆项目代码仓库**
    ```bash
    git clone 你的项目仓库地址
    cd 你的项目文件夹
    ```

2.  **配置CARLA仿真环境**
    - 下载并安装 [CARLA Simulator 0.9.14](https://github.com/carla-simulator/carla/releases/tag/0.9.14)
    - 进入CARLA安装目录，运行 `CarlaUE4.exe`，等待仿真服务端口 `2000` 启动完成，控制台输出`Server ready`即代表启动成功。

3.  **一键安装全部Python依赖**
    ```bash
    pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
    ```
    > 若安装失败，可手动逐个安装核心依赖：
    > `pip install pygame opencv-python numpy -i https://pypi.tuna.tsinghua.edu.cn/simple`

4.  **配置CARLA PythonAPI路径**
    项目代码已自动适配CARLA API路径，若出现`No module named 'carla'`错误，可手动将CARLA安装目录下的`PythonAPI/carla/dist`中的`.egg`文件路径添加到系统环境变量。

## 快速启动项目
### 1. 完整运行命令
在项目根目录执行主程序，支持自定义参数配置：
```bash
python src/your_main_file.py --host 127.0.0.1 --port 2000 --res 1280x720
```
- `--host`：CARLA仿真器IP，默认本地地址`127.0.0.1`
- `--port`：仿真器服务端口，默认`2000`
- `--res`：窗口分辨率，默认`1280x720`

### 2. 按键交互说明
| 按键 | 功能 | 说明 |
| :--- | :--- | :--- |
| `1/2/3/4/5` | 切换车辆视角 | 支持后视角、前视角、侧方视角、鸟瞰视角等多视角切换 |
| `C` | 切换下一种天气 | 循环切换晴天→雨天→雾天→黑夜，验证自适应功能 |
| `V` | 切换上一种天气 | 反向循环切换天气 |
| `ESC` | 退出仿真程序 | 安全退出并销毁仿真环境中的车辆与传感器 |

### 3. 运行时状态说明
程序启动后，控制台会实时打印识别结果与车辆状态：
```
当前天气：sunny
交通标志识别：none
红绿灯识别：green
前方行人：无
```
Pygame窗口会实时渲染车辆视角画面，左上角显示车辆速度、服务器FPS等信息，触发车道偏离时会弹出红色警告文字。

## 运行效果展示
1.  **基础行驶**：车辆启动后自动进入定速巡航模式，保持车道居中行驶，无人工干预即可稳定通过直道、弯道场景。
2.  **交通信号响应**：行驶至十字路口时，可识别红绿灯状态，红灯自动停车等待，绿灯自动起步通行，黄灯提前减速。
3.  **行人避让**：前方出现行人时，车辆立即触发紧急制动，完全刹停等待，行人离开后自动恢复巡航。
4.  **车道偏离预警**：车辆压线时，屏幕中央弹出“车道偏离警告！”红色大字，同时播放提示音提醒驾驶员。
5.  **天气自适应**：切换为黑夜/雨天时，车辆自动开启近光灯/雾灯，同时相机参数自动调整，提升画面可见度。

## 常见问题与调试说明
1.  **CARLA连接失败**：
    - 检查CARLA仿真器是否已启动，端口`2000`是否被占用
    - 确认Python版本与CARLA API兼容，推荐使用Python 3.7
2.  **行人/交通标志误检**：
    - 调整`pedestrian_detect`/`traffic_sign_detect`函数中的HSV阈值或轮廓面积，收紧检测条件
    - 缩小ROI区域，过滤道路外的背景干扰
3.  **车辆控制不稳定**：
    - 调整PID控制器的`Kp/Ki/Kd`参数，降低`Kp`可减少车辆抖动，提高`Ki`可提升抗坡度能力
