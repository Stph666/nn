IntelliCar
面向自动驾驶车辆、基于城市仿真环境的计算机视觉目标检测与识别技术应用
项目简介
本项目基于 YOLO 算法训练模型，完成图像目标检测任务；依托 CARLA 仿真平台开展仿真实验，并完成了与 ROS 机器人操作系统的工程集成开发。


所用工具
数据集：整合多款公开数据集，并补充多气候、多路况场景实拍图像，依托 Roboflow 平台制作项目专属自定义数据集。
Roboflow 平台数据集地址
仿真器：选用开源自动驾驶仿真软件 CARLA。
CARLA 官网
CARLA 开源仓库
模型训练：基于 Ultralytics 推出的最新版 YOLOv8 算法搭建检测模型。
YOLOv8 开源仓库
ROS 系统：部署模型推理预测模块与车辆辅助决策功能模块。
ROS-Bridge：实现 CARLA 仿真环境与 ROS 之间的数据通信交互。
ROS-Bridge 开源仓库
